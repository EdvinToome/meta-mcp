# Notion-Derived Workflow (Local Copy)

Source pages captured locally:
- Nano Banana 2 Prompt Document
- Master Brand Research Prompt
- Claude Follow-up Prompt
- Template Prompts

Core process:
1. Build brand DNA context first (visual + copy constraints).
2. Convert template prompt into brand-specific standalone generation prompt.
3. Keep each prompt self-contained and immediately runnable.
4. Produce high-converting static creative intended for ad delivery.

Master brand research guidance:
- Extract brand voice adjectives, positioning, and differentiation.
- Extract exact color and typography intent.
- Extract photography direction and mood.
- Extract product packaging fidelity constraints.
- Produce a short image-generation modifier to prepend to prompts.

Claude follow-up guidance:
- Fill template prompts with brand-specific detail.
- Fill product-specific details from the creative brief/facts.
- Return finished prompts that do not need manual edits.

Template families to use:
- headline
- offer_promotion
- testimonials
- features_benefits
- social_proof
- us_vs_them
- before_after_ugc
- press_editorial
- curiosity_gap
- ugly_postit

Quality constraints:
- conversion clarity over abstract style
- clear visual hierarchy
- mobile readability
- preserve product and brand identity
