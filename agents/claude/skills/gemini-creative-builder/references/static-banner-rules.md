# Static Banner Rules (Local Copy)

Use these rules in every prompt plan.

Mandatory:
- include at least one scroll-stop mechanism
- keep one dominant visual claim
- reserve an overlay-safe zone
- keep feed legibility at mobile size

Scroll-stop mechanisms:
- high visual contrast
- pattern disruption element
- strong emotion/expression
- visual tension/incompleteness
- unusual lighting/color treatment

Practical constraints:
- avoid gibberish text artifacts
- avoid malformed objects or anatomy
- avoid visual clutter
- prefer realistic textures and shadows

Mode rules:
- full mode: short overlay lines, strong contrast, large readable hierarchy
- visual-only mode: no letters/logos/word-like forms; preserve negative space
