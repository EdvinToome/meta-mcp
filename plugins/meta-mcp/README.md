# Meta Ads MCP Claude Plugin

This plugin installs:
- project slash commands for publish, reporting, diagnosis, and copy
- bundled Meta Ads skills and starter docs

Before using the workflows in a project:
1. Run `/meta-mcp-init`.
2. Fill `.claude/meta-mcp/site-profiles.local.json`.
3. Fill `.claude/meta-mcp/BUSINESS_RULES.local.md`.
4. Make sure the global `meta` MCP server is configured for Claude Code Desktop.
5. Follow `MCP_USAGE.md` so the agent uses live Meta tools first and only uses docs when needed.
